# README
基于node开发的爬虫项目,使用了cheerio插件进行解析DOM元素，eventproxy插件进行控制异步，superagent进行访问url
## 使用插件
```
    "cheerio": "^1.0.0-rc.3",
    "eventproxy": "^1.0.0",
    "express": "^4.17.1",
    "superagent": "^6.1.0"
```
## 项目环境
```
Node.js v12.18.2.
```
## 运行
```
node index.js
```
## 访问
```
localhost:3000
```
